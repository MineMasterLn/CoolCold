# Етап розробки
#### 🧠 Етап 0: Пре-продакшн та екзистенціальні роздуми
- [x] Раптова поява ідеї о 3-й ночі
- [x] Спроба пояснити ідею коту/друзям (нічого не зрозуміли, але підтримали)
- [x] Очікування натхнення (тривало довше, ніж сама розробка)
- [x] Малювання кривого креслення на звороті чека з магазину
- [x] Пошук вільного часу в розкладі (не знайдено, вирішено працювати замість сну)

#### ⚡ Етап 1: Електроніка та "Магія Синього Диму"
- [x] Вибір мікроконтролера (Arduino, ESP32 чи просто два дроти і молитва)
- [x] Підбір компонентів, що не горять від погляду
- [x] Замовлення запчастин (режим очікування кур'єра активовано)
- [ ] Розпаковка посилок (найкращий момент у житті)
- [ ] Написання базової прошивки (вона не компілюється)
- [ ] Фікс багів (тепер компілюється, але не працює)
- [ ] Перемога над кодом (воно моргнуло світлодіодом!)
- [ ] Героїчна пайка без вдихання каніфолі (майже вдалося)

#### 🏗️ Етап 2: 3D Моделювання (Fusion 360 & Сльози)
- [ ] Створення професійного креслення (нарешті в зошиті в клітинку)
- [ ] Відкриття Fusion 360 та п'ятихвилинний ступор перед пустим екраном
- [ ] Малювання першого куба (це вже перемога)
- [ ] Перенесення ідеї в 3D (модель виглядає як космічний корабель, а не корпус)
- [ ] Спроба зробити "скруглення кутів" (Fillet Error: The operation could not be completed)
- [ ] Підготовка файлу до друку (слайсинг, підтримки, надія на адгезію)

#### 🖨️ Етап 3: Битва з 3D принтером
- [ ] Натискання кнопки "Print" та затамування подиху
- [ ] Спостереження за першим шаром протягом 15 хвилин (гіпноз)
- [ ] Молитва богам PETG-пластику, щоб нічого не відклеїлось
- [ ] Боротьба з "соплями" та воблінгом
- [ ] Зняття деталі зі столу (разом з частиною покриття столу)

#### 🛠️ Етап 4: Фінальна збірка та логістика
- [ ] Впихання невпихуємого (електроніка в корпус)
- [ ] Закручування останнього гвинтика, який чомусь залишився зайвим
- [ ] Тестовий запуск (диму немає — це хороший знак)
- [ ] Етап "І так зійде" (косметичні правки синьою ізолентою)
- [ ] Упаковка в коробку, де більше пупирки, ніж самого виробу
- [ ] Доставка до кінцевого пункту призначення
- [ ] Офіційне святкування завершення (сон мінімум 12 годин)
